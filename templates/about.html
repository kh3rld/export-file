ASCII Art Generator

About ASCII Art Web Stylize

Welcome to the ASCII Art Web Stylize, an innovative tool that transforms text into creative ASCII art. Our web interface is designed to be both intuitive and visually appealing, making it easy for users to generate stunning ASCII art with minimal effort.

Key Features

  • Transform strings into detailed ASCII art
  • Supports a wide range of characters including letters, numbers, and special symbols
  • Multiple banner styles for diverse artistic representations
  • Interactive and user-friendly web interface

Getting Started

To use this tool, you need to have Go installed on your system. Follow these simple steps to get started:

  1. Clone the repository: 
    git clone https://learn.zone01kisumu.ke/git/khahussein/ascii-art-web.git
  2. Navigate to the project directory: 
    cd ascii-art-web-stylize/cmd/
  3. Start the server: 
    go run .

    Access the tool at http://localhost:8080

Testing

To test the application, run:

go test -v

For more interactive testing, use:

curl -X POST -d "Text=Hello World&Banner=standard" http://localhost:8080/ascii-art
